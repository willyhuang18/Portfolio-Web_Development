2. Important SEO Header Tags

SEO (Search Engine Optimization) is critical for accessibility. Here is what I learned from the readings:

  • <title>: The primary identity of the page for search engine crawlers.
  • <h1> to <h6>: Essential for content hierarchy and screen readers.
  • <meta name="description">: The "first impression" text in search results.
  • <link rel="canonical">: Prevents confusion by pointing to the "master" version of a page.

5. Font Size Comparison & Usage

As a Behavioral Science major, I know readability is key. Here is when I would use each unit:

  • 12px (Absolute): Best for copyright or legal disclaimers that must stay small.
  • 20px (Absolute): Useful for fixed-width sidebars where design shouldn't shift.
  • 1.1em (Relative): Perfect for standard body text that should scale with its parent.
  • 2rem (Relative): My choice for main headers to ensure they scale with root settings.
  • 140% (Relative): Great for blockquotes or emphasized notes.
  • 4vw (Relative): Ideal for hero banners that need to look big on all screen widths.

6. Positioning Creative Section

When to use:

  • Static: For normal document flow (like this text).
  • Relative: When I need to nudge an element without affecting others.
